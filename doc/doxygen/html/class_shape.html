virtual std::vector<double> Shape::intersect ( const Ray &  ray) const
pure virtual

Get the parametric values on the ray where the shape is intersected

A point on a ray can be written as O + tD, where O is the origin, D the direction and t a scalar. This function will return all values t where the ray hit the shape. If the ray did not hit the shape, the returned vector is empty.

Parameters
ray: Tested ray
Returns
a vector containing all values t where the shape is intersected
