std::vector<double> Sphere::intersect ( const Ray &  ray) const
overridevirtual

Get the parametric values on the ray where the sphere is intersected

A point on a ray can be written as O + tD, where O is the origin, D the direction and t a scalar. This function will return all values t where the ray hit the sphere. If the ray did not hit the sphere, the returned vector is empty.
