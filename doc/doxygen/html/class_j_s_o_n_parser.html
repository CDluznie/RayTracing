Parse a JSON file scene

The description file contains the scene itself and the used camera to record the scene. Then the function return a pair of the scene and the camera described in the file. The camera is defined in the file by a position and a target. The scene is defined in the file by a list of shapes, a list of lights and a background color. For more precision on the format read the README.MD file

Parameters
fname: File to parse
width: Width of the image
height: Height of the image
Returns
the scene and the camera described in the file
